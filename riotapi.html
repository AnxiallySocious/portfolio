Given my love for solved games, I wanted to solve some pieces of a popular video game, League of Legends.

Riot Games, who makes League of Legends, has a very generous API available to the public. With this project, I've focused on learning to navigate API codes, errors, call limits, and json-formatted databases.

While I'm still making the transition from the original java project to this website, I've so far been able to record millions of pieces of data, extrapolating winrate data based on team composition.
